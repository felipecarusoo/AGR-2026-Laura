# Projeto Agrinho 2026 - Estrutura Inicial

Este projeto esta em desenvolvimento para o Concurso Agrinho 2026, categoria Programacao (HTML, CSS e JavaScript).

## Status atual

- A pagina principal (`index.html`) e o foco atual de desenvolvimento.
- A estrutura de pastas ja foi preparada para facilitar a expansao para paginas secundarias.
- As pastas estao com placeholders (`.gitkeep`) apenas para manter a organizacao no repositorio.

```text
AGRINHO LAURA/

    img/
      autorais/

```

## Estrategia de desenvolvimento

1. Refinar completamente o `index.html` (layout, usabilidade, acessibilidade e interacoes).
2. Definir o design final das secoes com base na identidade visual da estudante.
3. Criar paginas secundarias em `pages/` quando o design de cada tema estiver aprovado.
4. Migrar CSS e JS para as pastas `css/` e `js/` gradualmente, sem quebrar o funcionamento.
5. Inserir imagens autorais em `assets/img/autorais/` com creditos documentados.
